SOBRE MÍ

Soy un desarrollador web con un enfoque en frontend y conocimientos básicos de backend. Mi formación en Ciencias de la Computación me ha permitido trabajar con metodologías, como esPrototipo y Scrum, lo que me facilita trabajar en equipos colaborativos y gestionar proyectos de manera organizada y eficiente. Utilizo tecnologías como HTML, CSS, JavaScript y Bootstrap para desarrollar sitios web, creando además prototipos y diseños de interfaces de usuario mediante Figma.

En el área de backend, tengo experiencia con PHP, MySQL y el framework CodeIgniter, así como conocimientos en Java y SQL para la gestión y manipulación de datos. Además, trabajo con el patrón MVC para estructurar aplicaciones de manera modular y escalable. Tengo experiencia en el uso de GitHub Desktop para el control de versiones y la gestión de repositorios, lo que facilita la colaboración en proyectos de desarrollo.

HABILIDADES

PROYECTOS

PANECITO

Es un sistema de punto de venta para una panadería, por medio de un CRUD permite agregar, editar o eliminar productos en stock, Cajero, Pedidos y Sucursales. Este proyecto colaborativo fue desarrollado utilizando HTML5, CSS3, JavaScript, PHP y MySQL.

DISREXIA

Es una aplicación web diseñada para ayudar a niños de 9 a 12 años que han sido diagnosticados con dislexia. La aplicación cuenta con dos módulos: lecciones y ejercicios. Este proyecto colaborativo fue desarrollado utilizando HTML5, CSS3 y JavaScript.

DISGRAFI

Disgrafi es un sistema de seguimiento y una herramienta interactiva diseñada para niños con disgrafía. Ofrece módulos de seguimiento, lecciones y ejercicios. Este proyecto ha sido desarrollado utilizando la metodología SCRUM, framework CodeIgniter, junto con HTML5, CSS3, JavaScript, Bootstrap, PHP y MySQL.

ZRUNNER7

Es una página web, desarrollada para la promoción de eventos deportivos, para dar infromación de las próximas ccarreras que habrá y asi fomentar el turísmo y el deporte.. Este proyecto esta desarrollado con HTML, CSS y JAVASCRIPT, CodeIgniter, Figma
